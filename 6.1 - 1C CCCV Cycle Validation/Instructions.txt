Instructions

To Get Results in Main Manuscript, Section 6.1:

1. Generate 1C CCCV Drive Cycle
-> Open MATLAB_CCCV_cycle_2
-> Set Param = 1 in line 8. To select the LCO Parameter Set
-> Run MATLAB_CCCV_cycle_2

2. Generate PyBaMM Validation Data
-> Open PyBaMM_Script_For_Validation_LCO
-> Run PyBaMM_Script_For_Validation_LCO

3. Generate Results for 1C CCCV Validation
-> Open MATLAB_PyBaMM_1C_CCCV_Comparison
-> Set Param = 1 in line 8. To select the LCO Parameter Set
-> Run MATLAB_PyBaMM_1C_CCCV_Comparison
